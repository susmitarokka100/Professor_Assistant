# Professor Assistant Project

This project is a Python program that helps professors create exams from a question bank.

## Features
- Reads question-answer pairs from a question bank file.
- Allows professor to choose how many questions to include.
- Randomly selects questions using `randint`.
- Saves the exam to an output file.
- Lets professor create multiple exams.

## Files
- `professor_assistant.py` → main Python program.
- `question_bank.txt` → contains questions and answers.
- `midterm.txt` → generated exam file (if created).
